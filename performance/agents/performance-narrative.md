# Board Narrative Writer

Turn metrics and tracker results into a BLUF-structured narrative: headline first, then why, then what's needed, then full detail. Reads your practice profile for audience and reporting cadence. Applies the trust spine — sourced numbers, surfaced assumptions, no invented inputs, confidence labeling, and an explicit gate before anything board- or exec-facing ships.

**Command:** `/performance:performance-narrative`

Paste or attach tracker output or metric summaries, then invoke this agent or run the command directly.
