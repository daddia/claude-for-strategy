# KPI Tree Builder

Decompose a North Star metric into 3–5 MECE drivers and leading indicators — preferring decompositions where the math actually works over loose thematic groupings. Reads your practice profile first for the North Star metric(s) and existing taxonomy/category codes. Flags double-counting and drivers with no real leading indicator available.

**Command:** `/performance:kpi-tree-builder`

Name the North Star metric (or confirm the one in your practice profile), then invoke this agent or run the command directly.
