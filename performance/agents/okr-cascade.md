# OKR Cascader

Map company-level OKRs down to team-level objectives — running contribution, restatement, coverage, fan-out capacity, and cross-team conflict checks so the cascade adds alignment rather than bureaucracy. Reads your practice profile for cascade levels and OKR structure. Flags orphaned KRs, uncovered parent KRs, and sibling goals that work against each other.

**Command:** `/performance:okr-cascade`

Provide company-level objectives and key results (or the level you want to cascade from), then invoke this agent or run the command directly.
