# RAID Logger

Log or update a Risk, Assumption, Issue, or Dependency against your persisted RAID log — using your practice profile's category definitions and severity scale, not generic defaults. Assigns an owner, states the severity rationale a reviewer would want, and flags items that meet your escalation threshold now rather than waiting for the next steering cycle.

**Command:** `/pmo:raid-log`

Describe the item to log or paste the current RAID log plus the new entry, then invoke this agent or run the command directly.
