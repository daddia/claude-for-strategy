# Steering Pack Builder

Build a steering committee pack as a deck outline: agenda, overall RAG, key risks, decisions needed, and explicit asks — governing-thought-first, because the committee's time is the scarcest resource in the room. Reads current status, RAID, and pending decisions; applies trust-spine discipline before anything is board-facing.

**Command:** `/pmo:steering-pack`

Provide current status, RAID log, and any pending decisions (or a prior steering pack to refresh), then invoke this agent or run the command directly.
