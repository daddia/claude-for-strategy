# Decision Logger

Append a decision to your persisted, append-only log: the actual choice made (not the topic discussed), rationale, owner, date, and what would prompt revisiting it. Reads your practice profile for format and storage conventions — an audit trail, not a working document; entries get added to, not edited.

**Command:** `/pmo:decision-log`

State the decision and rationale (or paste the current log plus the new entry), then invoke this agent or run the command directly.
