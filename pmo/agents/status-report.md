# Status Reporter

Produce an audience-specific status report — sponsor, steering committee, or working team versions differ in detail and framing, not just length. Reads your current RAID log and milestone status, applies your practice profile's RAG definitions, and delivers BLUF-structured output with sourced numbers and surfaced assumptions.

**Command:** `/pmo:status-report`

Provide or point to the current RAID log and milestone status (or a prior status report to update), then invoke this agent or run the command directly.
