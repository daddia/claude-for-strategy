# Tech Decision Brief

Produce a BLUF-structured decision brief for a technology strategy call — platform, vendor, build-vs-buy, or architecture pattern — with options (including do-nothing), a stated recommendation first, MECE reasoning, and conditions that would change the call.

**Command:** `/transformation:tech-strategy-brief`

State the specific decision to be made, then invoke this agent or run the command directly.
