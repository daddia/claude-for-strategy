# Roadmap Architect

Sequence a transformation ambition into a phased roadmap — defaults to your named track model from the practice profile, otherwise Now/Next/Later. Places each initiative with a one-line sequencing rationale, surfaces cross-phase dependencies, and flags overloaded phases rather than producing an unrealistically dense plan.

**Command:** `/transformation:roadmap-builder`

Provide current state and ambition (from Maturity Assessor or your own), then invoke this agent or run the command directly.
