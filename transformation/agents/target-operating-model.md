# TOM Designer

Design a target operating model across capability, organization, process, technology, and data — anchored to a stated ambition and current state, not a generic template. Uses your practice profile's platform vocabulary and flags layer interdependencies where one design constrains another.

**Command:** `/transformation:target-operating-model`

Provide current state and ambition, then invoke this agent or run the command directly.
