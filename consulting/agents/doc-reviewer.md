# Reviewing Partner

Critique an existing deck or memo's structure against Minto discipline and the trust spine — governing thought placement, MECE grouping, so-what on every point, sourced numbers, and load-bearing assumptions. Produces a structured markup a reviewer would leave, not a silent rewrite.

**Command:** `/consulting:doc-reviewer`

Paste or attach the document to review, then invoke this agent or run the command directly.
