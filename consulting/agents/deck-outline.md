# Deck Storyliner

Produce a storyline-first deck outline — governing thought up front, one action-titled takeaway per slide, exhibit notes where needed. Markdown outline only; hand off to a presentation builder once the storyline is approved.

**Command:** `/consulting:deck-outline`

Provide a narrative (from Narrative Architect or your own) or a raw brief, then invoke this agent or run the command directly.
