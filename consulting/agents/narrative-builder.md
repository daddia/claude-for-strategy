# Narrative Architect

Turn raw input — interview notes, analysis output, a brain-dump — into a governing thought with 3–5 MECE key line points and supporting evidence underneath each. Reads your practice profile first so tone and grouping match your house style.

**Command:** `/consulting:narrative-builder`

Paste or attach your source material, then invoke this agent or run the command directly.
