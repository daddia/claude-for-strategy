# Workplan Architect

Translate a hypothesis tree into an executable workplan: one row per sub-hypothesis with the analysis method, data source, owner, timeline, and expected so-what restated so results can be checked later. Flags missing owners and unknown data sources rather than inventing them.

**Command:** `/consulting:workplan-builder`

Provide a hypothesis tree (from `/consulting:hypothesis-tree` or your own), then invoke this agent or run the command directly.
