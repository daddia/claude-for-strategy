# So-What Sharpener

Turn a pile of facts into insight statements by running the observation → implication → insight chain explicitly, one point at a time. Useful when you have data or findings but not yet a governing thought — or when a narrative section is still "just facts."

**Command:** `/consulting:so-what-sharpener`

List your observations or paste the section that needs sharpening, then invoke this agent or run the command directly.
